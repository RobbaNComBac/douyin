低质量视频

	低质量，不知道表达什么

	或者模糊

	7秒或以下，不行



搬运/非原创视频

	去水印

	大V的视频，抖音保护，去水印也没用，不要混剪



隐形风险

	视频文案中出现欺诈的内容，包含极限词（最高级，神器，效果最好，标题党）

	野生动物，抽烟喝酒，疑似赌博



广告营销

	品牌logo，品牌词，品牌水印，口播

	个人的电话，微信，qq号，二维码

	商品价格的信息

		尽量不要讲价格



未授权明星/影视/赛事视频

	比如刚上映的电视剧，可能会火，但是被人举报，会有一个限流的操作。



视频购物车异常

	橱窗打不开或者下架了

	有一些优惠劵，或者营销类的，买一送一的营销词
